Contents of the ZIP File:

GeneticAlgorithm.java
Individual.java
Population.java 
OptimizeFunction.java
Report for Genetic Algorithm
Output files

----------------------------------------------------------------------------------------------------------------------
How to Run the Program:

1.Extract the ZIP File: Unzip the contents into a directory of your choice.

2.Compile the Java Code: Navigate to the directory in a terminal or command prompt and compile the Java files. 
For example:

javac GeneticAlgorithm.java Individual.java Population.java OptimizeFunction.java

3.Run the Main Class: Execute the OptimizeFunction class to start the optimization process:

java OptimizeFunction
----------------------------------------------------------------------------------------------------------------------
Configuration Parameters:

The GA parameters can be adjusted in OptimizeFunction.java. 

These include:

Population size
Mutation rate
Crossover rate
Number of generations
Elitism count
